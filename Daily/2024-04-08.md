### 导览分组的封面图

分组的封面图不直接设置，使用其导览节点的第一个的缩略图。

```ts
export interface TourInfo extends SpeechInfoReplace, VC3DConfig {
    /**
     * 导览名称
     */
    name?: string;
    /**
     * 缩略图
     */
    thumbnail?: string;
    /**
     * 导览id
     */
    id: string;
    /**
     * ...
     */
}
```

导览的缩略图创建与更新场景封面类似，使用 `createScreenShot` 来创建当前 Canvas 的截图。
