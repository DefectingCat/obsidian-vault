Rust 的数组类型是存储在栈内存中的，但是它也是可以存储非基础类型的值。不过它不可以使用这样的数组初始化语法：

```rust
let a = [3; 5];
```

`let a = [3; 5];` 的本质是 Rust 不断的 Copy 出来的，而非基本类型是无法深拷贝的。

正确的做法应该是使用 `std::array::from_fn` 。

```rust
fn main() {
    let arr: [String; 6] = core::array::from_fn(|i| format!("Arr {}", i));
    println!("{:?}", arr);
}
```
