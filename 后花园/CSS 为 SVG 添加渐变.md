其中，需要为 `linearGradient` 设置一个 `id`。然后再通过 CSS 的 ID 选择器，来为其颜色变量设置颜色。

```css
    .heart {
      margin-right: 0.04rem;

      #heart-gradient {
        --color-stop-one: #9330e5;
        --color-stop-two: #1067f7;
      }
    }
```
